## 範圍

Server 領域收錄所有與**伺服器運維、自動化管理、系統工具**相關的實作筆記。範圍刻意寬一點，讓相鄰技術（PowerShell、Bash、systemd、Docker 等）能在同一棵知識樹下被找到。

## 子領域

- **[PowerShell](./powershell/)** — Microsoft 的物件導向 shell；本站從這裡開始累積。

> 之後預期擴充：Bash、Linux 服務管理、Docker、SSH 與遠端管理。新增 subdomain 時遵守 [知識組織原則](/meta/principles/)。

## 學習路徑建議

1. 先依目前主要工作平台選擇 subdomain（例如日常用 PowerShell 就先進 [PowerShell](./powershell/)）。
2. 在 subdomain 內挑一個 topic 作為入門（從 `basic-learn` 開始最穩）。
3. 依 topic → skill → concept 一路深入，每個 concept 一頁。
